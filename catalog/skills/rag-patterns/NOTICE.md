## NOTICE of Modification

**The original source material consisted of 21 Python application tutorials
demonstrating various RAG patterns (awesome-llm-apps), a graph-based RAG system
with knowledge graph construction and community detection (microsoft/graphrag),
and a vectorless reasoning-based RAG framework with hierarchical tree indexing
(VectifyAI/PageIndex). These have been synthesized by Yunseo Kim on 12026-02-25
into a unified instructional skill guide. The synthesis extracted architectural
patterns, retrieval strategies, and best practices from the implementations
while removing application-specific code, UI logic, and API key management.**
