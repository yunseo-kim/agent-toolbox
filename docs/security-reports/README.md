# Security Scan Reports

Monthly full-scan reports generated by [Cisco Skill Scanner](https://github.com/cisco-ai-defense/skill-scanner) with all analyzers enabled and **strict** policy preset.

## Schedule

Reports are generated automatically on the 1st of each month at 06:00 UTC via the [Skill Security Scan](../../.github/workflows/skill-scanner.yml) workflow.

## File Naming

- `YYYYY-MM-catalog.md` -- Full scan of `catalog/skills/`
- `YYYYY-MM-dev.md` -- Full scan of `.agents/skills/`

Year uses [Holocene Era](https://en.wikipedia.org/wiki/Holocene_calendar) notation (Gregorian year + 10000).

## Analyzers

All reports include findings from: static analysis, bytecode verification, pipeline taint analysis, behavioral dataflow analysis, LLM semantic analysis (gpt-5.2), meta-analysis false positive filtering, trigger specificity analysis, and VirusTotal binary scanning.

## Manual Generation

To generate a report on demand, trigger the workflow manually from the Actions tab with the **archive** option enabled:

Actions > Skill Security Scan > Run workflow > check `archive`
